# Advanced Todo App

A feature-rich Todo application built with React for the frontend and Node.js. This app leverages local storage for persistent data storage, allowing users to manage tasks even when offline.

## Features

- **Add, Edit, and Delete Todos**: Easily manage your tasks with intuitive controls.
- **Local Storage**: Todos are saved locally to ensure persistence across sessions.
- **Mark Complete/Incomplete**: Keep track of tasks by marking them as complete or incomplete.
- **Filter Todos**: Filter tasks based on status (all, completed, or pending).
- **Responsive Design**: Fully responsive UI for optimal experience on all devices.

## Tech Stack

- **Frontend**: React, CSS (for styling)
- **Backend**: Node.js
- **Storage**: Local Storage (Browser-based storage)

### Prerequisites

Make sure you have **Node.js** and **npm** installed.
